# Canvas Design

## Activation
When the user asks to create a poster, visual art, design composition, or any static visual piece as PNG or PDF.

## Workflow

### Step 1: Design Philosophy Creation

Create a VISUAL PHILOSOPHY — an aesthetic movement that will be expressed visually. Output as .md file.

**Name the movement** (1-2 words): e.g., "Brutalist Joy", "Chromatic Silence", "Metabolist Dreams"

**Articulate the philosophy** (4-6 paragraphs):
- Space and form
- Color and material
- Scale and rhythm
- Composition and balance
- Visual hierarchy

**Essential guidelines:**
- Avoid redundancy — each design aspect mentioned once
- Emphasize craftsmanship repeatedly — "meticulously crafted", "painstaking attention", "master-level execution"
- Leave creative space for interpretive choices

**Philosophy Examples:**
- "Concrete Poetry": Monumental form, bold geometry, Polish poster energy meets Le Corbusier
- "Chromatic Language": Color as primary information system, Josef Albers meets data visualization
- "Analog Meditation": Paper grain, ink bleeds, vast negative space, Japanese photobook aesthetic
- "Organic Systems": Rounded forms, natural clustering, modular growth patterns
- "Geometric Silence": Grid-based precision, Swiss formalism meets Brutalist material honesty

### Step 2: Deduce the Subtle Reference

Identify the conceptual thread from the original request. The reference must be:
- A subtle, niche reference embedded within the art
- Refined so it enhances depth without announcing itself
- Like a jazz musician quoting another song — only those who know will catch it

### Step 3: Canvas Creation

Express the philosophy on a canvas:

- Create museum or magazine quality work
- Use repeating patterns and perfect shapes
- Treat as scientific observation — dense accumulation of marks, repeated elements, layered patterns
- Add sparse, clinical typography and systematic reference markers
- Limited color palette that feels intentional and cohesive
- Embrace analytical visual language to express ideas about human experience

**Text as contextual element:**
- Always minimal and visual-first
- Let context guide scale — a punk venue poster may have aggressive type
- Nothing falls off the page, nothing overlaps
- Every element contained within canvas boundaries with proper margins
- Use different fonts — search for appropriate typefaces

**Anti-AI-slop rules:**
- Avoid excessive centered layouts
- Avoid purple gradients
- Avoid uniform rounded corners
- Avoid Inter font
- Never cartoony or amateur

### Step 4: Refinement Pass

Take a second pass to refine and polish:
- Do NOT add more graphics
- Refine what exists to make it crisp
- Respect design philosophy and minimalism
- Ask: "How can I make what's already here more of a piece of art?"

### Multi-Page Option
When requested, create additional pages along the same philosophy but distinctly different. Bundle in same PDF or multiple PNGs. Treat as a coffee table book — each page a unique twist telling a story tastefully.

## Output
- Design philosophy as .md file
- Final artwork as .pdf or .png file
- Both files delivered together
